{WindsOfChange}%
{Open-Source Lagrangian Particle Simulation Library (R/C++)}%
{Aug 2024 — Present}%
{}%
{%
\begin{itemize}
  \item Developed C++ library implementing Lagrangian particle dispersion,
  atmospheric turbulence, and fluid flow over complex hilly terrain with forest
  canopy, based on analytical models (Massman \& Weil 1999; Finnigan \& Belcher
  2004).
  \item Parallelized particle tracking using RcppParallel and OpenMP; designed
  for scalability to tens of thousands of particles across kilometer-scale
  domains.
\end{itemize}

{Open-Source Physics Simulation Library}%
{Jan 2024 — Sep 2024}%
{}%
{%
\begin{itemize}
  \item Multiscale/multiphysics simulation toolkit combining Rust and Python
